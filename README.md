# Password Generator
---

## Functions include:

- Identifies entries within 8 - 128 characters
- Lowercase Letters
- Uppsercase Letters
- Special Characters
- Numerical Characters

Input outside of the length parameters will be met with an error notification to try again that also specifies what went wrong.
